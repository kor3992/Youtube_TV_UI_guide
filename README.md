-Full Guide Chrome/Chromium-


1. Download Google Chrome / Chromium

2. Go to The youtube website

3. Open the DevTools With F12

4. Go to the small "Toggle device toolbar" icon in the left upper corner

5. Go to "dimensions" and choose "Edit..." 

6. Go to "Add custom device"

7. Put in the device name "RokuTV"

8. Put "Roku/DVP-15.2" into the User agent string

9. Click on "Add" (Not: "add custom device"(it will reset your configs))

10. Then go to "Dimensions" again and choose your profile (In this case "RokuTV")

11. While youre still in the devtools visit "https://youtube.com/tv#"

12. Once the Site finished loading you can exit the DevTools and make it fullscreen to get the ultimate experience

13. To exit this mode simply untoggle "Device Toolbar" and visit normal youtube

NOTE: Refreshing the site will reset your configurations
